## 主要特色、功能
```
1.可以不用開遊戲輸入指令&聊天  
2.可以透過私訊Bot來執行命令(須設定白名單)  
3.白名單指令系統(避免有心人士亂搞)  
4.自動接受tpa & tpahere(需白名單)
```
